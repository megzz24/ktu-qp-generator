MODULE 1: C Fundamentals and Control Statements
--------------------------------------------------
  - Character set, constants, identifiers, keywords

  - Operators, precedence, bitwise operators, expressions
  - Input and output statements
  - Structure of a C program, simple programs
  - Control statements: if, if-else, nested if, switch
